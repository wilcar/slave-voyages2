6.1 Tableau d’effectifs

Il s’agit de compter, pour chacune des valeurs de la variable yearam, le nombre d’observations ayant cette valeur. Ce « nombre de fois » s’appelle l’effectif. La construction d’un tableau d’effectifs appartient à la catégorie des opérations groupées dans Dplyr et s’effectue par le biais de la fonction count. La colonne d’effectifs est automatiquement appelée n.

voyages %>%
  count(yearam)

6.4 Représentations graphiques

6.4.1 Représentation graphique en base R

Vous pouvez “piper” la fonction plot après avoir compté le nombre d’observations par années. Le résultat obtenu est un graphique de type “nuage de points”.

voyages %>%
  count(yearam) %>%
  plot()

Vous pouvez personnaliser le graphique par une géométrie de type marche d’escalier (t="s") ou une courbe (t="l") et le compléter par un habillage dont la taille des caractères est réglée par la fonction cex.

voyages %>%
  count(yearam) %>%
  plot(t="s",                                                         
      main="Chronologie des expéditions de traite atlantique",
      sub="Voyages, The TransAtlantic Salve Trade Db",
      xlab="années",
      ylab="expéditions",
      cex.lab=0.80,
      cex.sub=0.75)

6.4.2 Représentations avancées (ggplot2)

Vous pouvez utiliser le package ggplot2 pour régler les intervalles des axes continus (fonction scale_x_continuous) et modifier l’apparence globale du graphique à l’aide nombreux thèmes mis à disposition (ici nous utilisons le thème “minimal”.

n_exped <- voyages %>%
  count(yearam)

gg_chrono1 <- ggplot(n_exped, aes(x= yearam, y= n))
gg_chrono1 +
  geom_step (color="#ff1744") +
  scale_x_continuous(breaks=seq(1500, 1875, by=25)) +
  labs(title = "Chronologie des expéditions de traite atlantique",
       caption ="Source : The Trans-Atlantic Slave Trade Database (2019).Variable yearam", x="",
       y="expéditions") +
  theme_minimal() +
  theme(plot.caption =element_text(colour =  "grey25"))

Vous pouvez animer votre graphique à l’aide du package gganimate et rajouter l’instruction +transition_reveal(yearam).

library(gganimate)

          caption ="Source : The Trans-Atlantic Slave Trade Database (2019).Variable yearam", x="",
       y="expéditions") +
  theme_minimal() +
transition_reveal(yearam)

Vous pouvez également affichez les principaux pics et vallées en employant le package complémentaire ggpmisc. Les arguments ignore_threshold et span permettant de sélectionner les variations les plus significatives. Le chevauchement des étiquettes est réglé par l’argument geom ="label_repel" en lien avec le package ggrepel.

library(ggpmisc)
library(ggrepel)

n_exped_pics_vallees <- voyages %>%
  count(yearam)

gg_chrono1 <- ggplot(n_exped_pics_vallees , aes(x= yearam, y= n))
gg_chrono1 +
  geom_step (color = "grey60") +
  scale_x_continuous(breaks=seq(1500, 1875, by=25)) +
  stat_peaks(ignore_threshold = 0.4, colour = "#ff1744", span =15, segment.colour = "black", size = 3) +
  stat_peaks(ignore_threshold = 0.4, colour = "#ff1744", span =15, geom ="label_repel", size= 3) +
    stat_valleys(ignore_threshold =- 0.1, colour = "#00796b", span =15, size = 3) +
  stat_valleys(ignore_threshold =- 0.1, colour = "#00796b", span =15, geom ="label_repel", size= 3) +
  labs(title = "Chronologie des expéditions de traite atlantique",
       caption ="Source : The Trans-Atlantic Slave Trade Database (2019).Variable yearam", x="",
       y="expéditions") +
  theme_minimal() +
  theme(plot.caption =element_text(colour =  "grey25")) +
  theme(panel.grid.major = element_blank(), panel.grid.minor = element_blank())
