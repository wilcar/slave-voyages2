Explorer la Trans-Atlantic Slave trade Database avec R

Chapter 4 Préalables

Ce document présente les notions de R spécifiques à la manipulation et l’analyse de base de séries temporelles. Une série temporelle ou chronologique (nous utiliserons les deux termes indifféremment) correspond à une série d’observations réalisées au cours du temps.

4.1 Packages

4.1.1 Tidyverse

  • dplyr : grammaire de manipulation des tableaux de données.
  • stringr : manipulation de chaînes de caractères.
  • ggplot2 :graphiques avancés.

4.1.2 Autres packages

  • ggpmisc, ggrepel, gganimate : extensions du package ggplot2.

4.1.3 Installation

Les packages peuvent-être installés via l’interface graphique de RStudio ou la commande :

install.packages("")

4.2 Appel des packages

La fonction library permet de charger un package et de rendre ses fonctionnalités disponibles. Elle doit être exécutée à chaque session.

library(tidyverse) 
library(ggpmisc)
library(ggrepel)
library(gganimate)
